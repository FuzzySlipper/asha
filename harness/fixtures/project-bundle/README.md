## voxel-durability.txt

The save -> compaction -> reload durability evidence for the canonical voxel edit
sequence:

1. `postLoad` is the generated baseline.
2. `postEdit` is after applying the authored edit log.
3. `postReload` reconstructs compacted snapshots plus retained edits.

`postEdit == postReload` is the durability guarantee. `postLoad != postEdit`
proves the fixture actually exercises persistence instead of round-tripping an
unchanged state.
